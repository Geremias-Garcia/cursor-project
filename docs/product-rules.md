# Product Rules

## Auctions

- Auctions have start and end dates.
- Auctions may extend automatically if bids happen near the end.
- Auctions cannot accept bids after ending.
- Minimum bid increment is configurable.
- Users cannot bid on their own auctions.

## Users

- Users can create auctions.
- Users can watch auctions.
- Users receive notifications.

## Bidding

- All bids must be transactional.
- Highest bid wins.
- Bid history must be immutable.
- Concurrent bids must be handled safely.
